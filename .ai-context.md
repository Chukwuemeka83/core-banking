### Multi-Tenancy Strategy

**Implementation:**
- `tenant_id` field in all projections (read models)
- Tenant context in JWT claims
- Middleware extracts tenant from auth
- All queries filtered by tenant_id
- Event metadata includes tenant_id

**Example:**
```go
// Extract tenant from context
tenantID := GetTenantFromContext(ctx)

// Query with tenant isolation
db.Where("tenant_id = ? AND user_id = ?", tenantID, userID).Find(&accounts)
```
